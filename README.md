# avant-marketplace-security
NIST CSF 2.0 Cybersecurity Plan for an E-Commerce SMB.

**Author:** Ahmad Mukhtar
**Framework Baseline:** NIST CSF 2.0
**Target Organization:** Simulated E-Commerce SMB (25 Employees)

---

## 📋 Project Overview
This project establishes a comprehensive cybersecurity posture for a simulated E-Commerce Small and Medium-sized Business (SMB) utilizing the **NIST Cybersecurity Framework (CSF) 2.0**. The goal is to identify, protect, detect, respond to, and recover from digital threats peculiar to online retail environments, ensuring data integrity and customer trust.

## 🗂️ Repository Structure
* `/docs` - Contains detailed policy documentations, risk assessments, and implementation guidelines.
* `README.md` - Core project introduction, baseline configuration, and metadata.

## 🛡️ Methodology
This plan aligns organizational security practices across the core functions of NIST CSF 2.0:
1. **Govern (GV) & Identify (ID):** Establishing asset management and organizational risk tolerance baselines.
2. **Protect (PR):** Implementing identity management, access control, and data security safeguards.
3. **Detect (DE):** Configuring monitoring systems to surface potential security events.
4. **Respond (RS) & Recover (RC):** Documenting incident response structures and backup resilience protocols.
